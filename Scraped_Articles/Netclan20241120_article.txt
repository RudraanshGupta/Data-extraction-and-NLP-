If a user clicks on tables values, then the bubble chart on the map should be highlighted that relates to the table. Project Description“I have a map visual. I would like to dynamically change the colours of some of the bubbles.”The report page has several filters and KPI Dashboard, whose metrics change dynamically when the user clicks a certain element.
